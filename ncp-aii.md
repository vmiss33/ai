# NVIDIA-Certified Professional AI Infrastructure (NCP-AII)

Study Resources

Offical Cert Page, download Exam Guide here: https://www.nvidia.com/en-us/learn/certification/ai-infrastructure-professional/

Master list of documents referenced in exam gude:

**NVIDIA DGX Systems User Guide**

- For recent systems: [NVIDIA DGX H100/H200 System User Guide (web, with PDF option)](https://docs.nvidia.com/dgx/dgxh100-user-guide)
- Newer example: [NVIDIA DGX GB200 System User Guide PDF](https://docs.nvidia.com/dgx/dgxgb200-user-guide/dgxgb200-user-guide.pdf)
- Another variant: [NVIDIA DGX B300 User Guide PDF](https://docs.nvidia.com/dgx/dgxb300-user-guide/dgxb300-user-guide.pdf) (Search docs.nvidia.com/dgx for your specific DGX model; the guide is core and repeated often.)

**NVIDIA Base Command Manager Documentation**

- [User Manual PDF (v11)](https://docs.nvidia.com/base-command-manager/manuals/11/user-manual.pdf)
- [Installation Manual PDF (v11)](https://docs.nvidia.com/base-command-manager/manuals/11/installation-manual.pdf)
- [Developer Manual PDF (v11)](https://docs.nvidia.com/base-command-manager/manuals/11/developer-manual.pdf)
- Main hub: https://docs.nvidia.com/base-command-manager/index.html (PDFs linked per section).

**NVIDIA Data Center GPU Manager (DCGM) User Guide**

- [DCGM User Guide PDF (v2.2)](https://docs.nvidia.com/datacenter/dcgm/2.2/pdf/DCGM_User_Guide.pdf)
- Older/reference version: [DCGM User Guide PDF (v1.7)](https://docs.nvidia.com/datacenter/dcgm/1.6/pdf/dcgm-user-guide.pdf)
- Main docs: https://docs.nvidia.com/datacenter/dcgm/latest/user-guide/index.html

**NVIDIA DGX OS Documentation** This is typically embedded in the DGX User Guides above (e.g., software stack sections). No standalone PDF, but covered in DGX system guides.

**NVIDIA AI Enterprise Documentation** Licensing/planning: [NVIDIA Enterprise Licensing Guide (web)](https://docs.nvidia.com/ai-enterprise/planning-resource/licensing-guide/latest/licensing.html) (PDF export available via browser print). Broader docs hub: https://docs.nvidia.com/ai-enterprise/ (no single master PDF; use search for specifics like deployment guides).

**NVIDIA InfiniBand / Spectrum-X Networking Documentation** No single PDF; refer to NVIDIA Networking docs (e.g., Cumulus Linux, MLNX_OFED). For AI infra focus, see DGX SuperPOD guides below.

**NVIDIA DGX SuperPOD Reference Architecture / Design Guide**

- [NVIDIA DGX SuperPOD Data Center Design (DGX H100) PDF](https://docs.nvidia.com/nvidia-dgx-superpod-data-center-design-dgx-h100.pdf)

**NVIDIA BlueField DPU Documentation** Primarily web-based at https://docs.nvidia.com/networking/display/BlueField (PDFs per component, e.g., user guides).

**NVIDIA Power and Cooling Best Practices** Often in SuperPOD design guides (linked above) or separate whitepapers; no dedicated standalone PDF found in searches, but check DGX/SuperPOD docs for facility sections.

**NVIDIA NGC Catalog Documentation** Web-based: https://docs.nvidia.com/ngc/ (no central PDF; container/user guides are online).
